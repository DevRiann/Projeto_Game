# Jogo de Matemática.JAVA

Como TCC fizemos essa aplicação em Java básico na implementação de em de um jogo de matemática

## Objetivo ##
Um game para testar o conhecimento de matemático do usuário com sistema de dificludade e pontuação.

## Estrutura ## 
* Declaração de variáveis privadas (Private) com acessores Getters;
* Estrutura de condional - IF/ELSE e SWITCH;
* Estrura de loop - While e FOR
* Estrtura de tratamento de erro - Try/Catch para implemntar o timeout no tempo de resposta;
